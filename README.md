# Cocktail App

Application for displaying cocktails using The Cocktail DB API.

## Installation and Setup

```bash
# Install dependencies
npm install

# Run in development mode
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Check code
npm run lint

# Format code
npm run format
```
